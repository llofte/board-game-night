- notes= (optional; spaces → +, commas → %2C)

Placing rules:
- If the user explicitly names a winner or finishing order, use that.
- If NO winner is mentioned, assume the order the players are listed IS the finishing order (1st, 2nd, 3rd...). Apply first= second= third= accordingly.
- Only ask about placing if there is genuine ambiguity (e.g. "it was close" with no order given).

Date rules:
- "tonight" or "today" = today's date
- "last night" = yesterday
- "Friday" = most recent Friday
- No date = yesterday

Notes rules:
- Extract any contextual details from the message — first-time players, rematches, unusual circumstances, how the game went, etc.
- Rewrite them as a clean, concise note (not verbatim). E.g. "note that it was Jay's first time playing" → "Jay's first time playing." Multiple details → separate with semicolons.
- If an expansion was used, lead with "w/ [Name] expansion" and append any other notes after a semicolon.
- URL-encode the final note string (spaces → +, commas → %2C, apostrophes → %27, semicolons → %3B).

Name rules:
- "Loftes" or "the Loftes" = Dave and Leanna (last name Lofte)
- "Harrells" or "the Harrells" = Ben and Shannon (last name Harrell)
- "Pavones" or "the Pavones" = Marc and Heather (last name Pavone)
- Accept close misspellings and nicknames
- Include any name mentioned even if not a regular player
- Always use first names only for players in the URL (e.g. p1=Dave, not p1=Dave+Lofte)
